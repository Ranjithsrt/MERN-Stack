Different Ways to Add CSS Styles
  1.Inline CSS
    CSS is written directly inside an HTML element using the style attribute.
    We cant use in larger project.

  2.Internal CSS
    CSS is written inside a <style> tag within the <head> section of an HTML file.
    Good for single-page styling but not reusable across multiple pages.

  3.External CSS
  CSS is written in a separate .css file and linked to the HTML file using a <link> tag in the <head>.
  Best for larger projects as it keeps styles separate, reusable, and easier to manage.
