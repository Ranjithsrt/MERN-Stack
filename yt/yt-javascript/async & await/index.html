Client and Server

What is HTTP?

HTTP - HyperText Transfer Protocol

It's the language that your browser and the server use to talk to each other.

  • GET - Read Data
  • POST - Create New Data
  • PUT - Update entire resource
  • PATCH - Update only some fields
  • DELETE - Delete data

HTTP Status Codes

  • 200 - OK
  • 201 - Created
  • 202 - Accepted
  • 300 - Multiple Choices
  • 301 - Moved Permanently
  • 400 - Bad Request
  • 401 - Unauthorized
  • 403 - Forbidden
  • 404 - Not Found
  • 500 - Internal Server Error
