## 🚀 HOW TO GET STARTED

### Step 1: Start Your App

```bash
cd my-app
npm run dev
```

### Step 2: Test the Checkout Flow

1. Go to `http://localhost:3000/products`
2. Add items to cart
3. Click "Proceed to Checkout"
4. Fill in shipping details (all 8 fields)
5. Select a shipping method
6. Click "Continue to Confirmation"
7. Review your order
8. Select a payment method
9. Click "Place Order"
10. See your order confirmation

### Step 3: View Your Orders

1. Navigate to `http://localhost:3000/orders`
2. See your order in the table
3. Click "View Details" to see full information
4. Check the status timeline

### Step 4: Check the Data

1. Open Developer Tools (F12)
2. Go to Application → LocalStorage
3. Look for `orders`, `shippingInfo`, `cart_*` keys
4. See your saved data

## 💾 DATA STORAGE

All data is saved in your browser's **localStorage**:

**Temporary (During Checkout):**

- `shippingInfo` - User address & contact
- `shippingMethod` - Selected method
- `shippingCost` - Shipping fee

**Permanent (After Order):**

- `orders` - Complete order history
- `cart_[userId]` - Shopping cart (clears after order)
