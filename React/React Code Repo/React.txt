Day-1

React.js (commonly called React) is a JavaScript library used to build fast, interactive, and component-based user interfaces, especially for single-page applications (SPAs).

It was developed by Facebook (Meta) on 2011 and first released in 2013 (Open Source).


Why React?
    - Build reusable UI components
    - Make fast applications using a virtual DOM
    - Manage data and views easily
    - Create single-page applications (SPA)
    - Work with other tools and libraries (Router, Redux, Axios, etc.)

Advantages of React
    - Fast performance
    - Component-based structure
    - Reusable code
    - Easy to build mobile apps using React Native
    - Strong community support

Where React is Used?
    - Facebook
    - Instagram
    - Netflix
    - WhatsApp Web
    - Airbnb
    - Uber

Prerequisite Knowledge to Learn ReactJS
    - HTML
    - CSS
    - JavaScript (Advance Knowledge) ES6
        - let, const, arrow functions
        - Template literals
        - Destructuring
        - Spread / Rest operators
        - Modules (import/export)
        - Promises, async/await
        - Ajax / Fetch

📁 React (Vite) Project :

📁 node_modules

All installed libraries and dependencies.
(You never edit this folder.)

📁 public

Static files that don’t change (images, icons, etc.).
index.html usually lives here (in Vite it’s in root).

📁 src

Your actual React code:

Components

CSS

App.jsx

main.jsx

This is the folder you work in.

.gitignore

Tells Git which files/folders to ignore (like node_modules).

eslint.config.js

Rules for checking code quality (optional).

index.html

Main HTML template Vite uses to inject React.

package.json

Project information + list of dependencies + scripts.
Most important file after src.

package-lock.json

Exact versions of installed libraries.
Helps reproducible installs.

README.md

Notes/instructions for the project.

vite.config.js

Vite configuration (plugins, server settings).

src - components  -

A.jsx - List Data

import React from "react";

const A = ({ items }) => {
  return (
    <div>
      <ul>
        {items.map((item) => (
          <li key={item.id}>{item.name}</li>
        ))}
      </ul>
    </div>
  );
};

export default A;

-----------------------------------

B.jsx - calculation 

import React from "react";

const B = ({ items }) => {
  const male = items.filter((item) => item.gender === "Male").length;
  console.log(male);
  return (
    <div>
      <p>Total Male : {male}</p>
      <p>Total Female : {items.length - male}</p>
    </div>
  );
};

export default B;
