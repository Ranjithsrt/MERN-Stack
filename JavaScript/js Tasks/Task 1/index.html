Task : 1 Inventory Management System


        This project simulates a small inventory or store catalog. You'll use an array where each item is an object.
        
        // Array of Objects
        let inventory = [
        { id: 101, name: "Laptop", price: 1200, stock: 5 },
        { id: 102, name: "Mouse", price: 25, stock: 50 },
        { id: 103, name: "Monitor", price: 300, stock: 12 }
        ];
        
        Required Functions:
        
        - addItem(id, name, price, stock) : Adds a new item object to the inventory array.
        - removeItem(id) : Finds an item by its id and removes it from the array (using filter() or splice()).
        - updateStock(id, quantity) : Finds an item by id and updates its stock property.
        - findPrice(id) : Finds an item by id and returns its price.
        - listAllItems() : Iterates over the array and logs all item details to the console (using forEach() or map()).
