Task : 2 Simple Student Gradebook 
        This project models a set of students and their scores, requiring more complex lookups and calculations.
        
        // Object where keys are student IDs and values are objects
        let gradebook = {
        "S001": {
        name: "Rajesh",
        courses: [
        { name: "Math", score: 92 },
        { name: "Science", score: 85 }
        ]
        },
        "S002": {
        name: "Sam Kumar",
        courses: [
        { name: "Math", score: 78 },
        { name: "Science", score: 95 }
        ]
        }
        };
        
        Required Functions
        
        - addCourseScore(studentId, courseName, score): Adds a new course object to a student's courses array.
        - getStudentAverage(studentId): Calculates and returns the average score for a single student (requires iterating over
        the student's courses array).
        - getCourseAverage(courseName): Calculates and returns the average score for all students in a specific course (requires
        iterating over the keys of the gradebook object).
        - listTopStudent(): Determines which student has the highest overall average.
