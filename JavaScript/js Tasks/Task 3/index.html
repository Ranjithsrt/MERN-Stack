Task : 3* Contact Book and Search Utility (Focus on Array Searching/Sorting)

        This project focuses on manipulating an array of simple contact objects to implement search and organization features.
        
        // Array of Objects
        let contacts = [
          { firstName: "Ram", lastName: "Kumar", phone: "9859785895", email: "ram@example.com" },
          { firstName: "Sam", lastName: "Sundar", phone: "7859685785", email: "sam@example.com" },
          { firstName: "Ravi", lastName: "Teja", phone: "7858958896", email: "ravi@example.com" }
        ];
        
        *Required Functions*
         - *addContact(fName, lName, phone, email)*: Creates a new contact object and pushes it to the contacts array.
         - *searchByLastName(lastName)*: Uses the array's filter() method to return an array of contacts whose last name matches
        the input.
         - *sortByFirstName()*: Uses the array's sort() method to arrange the contacts alphabetically by their firstName.
         - *updateEmail(phone, newEmail)*: Finds the contact with the matching phone number and changes their email property.
