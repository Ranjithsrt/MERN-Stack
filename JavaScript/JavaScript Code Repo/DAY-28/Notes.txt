Document Object Model (DOM)

1.Accessing DOM Elements
  - getElementById(id)              # Single Element
  - getElementsByClassName(class)   # Multiple Elements
  - getElementsByTagName(tag)       # Multiple Elements
  - querySelector(selector)         # Single Element
  - querySelectorAll(selector)      # Multiple Elements

2.Modifying Content and Attributes
  - Text Content  ->   textContent
  - HTML Content  ->   innerHTML 
  - Attributes    ->   getAttribute() / setAttribute()
  - Styles        ->   style property (e.g., element.style.color = 'red';).
